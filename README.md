# Portfolio & CV Técnico - Christian Nuñez

Este repositorio contiene el código fuente de mi CV web profesional desarrollado en HTML5 y CSS3, orientado a posiciones Junior o Trainee como Desarrollador Backend Java.

---

## 📄 Descripción

Se trata de una página web responsive donde presento mi perfil profesional, stack tecnológico, formación académica y proyectos técnicos desarrollados durante mi formación en la Licenciatura en Informática.

---

## 👨‍💻 Perfil Profesional
Soy estudiante intermedio de la **Licenciatura en Informática (UNAHUR)**. Mi enfoque principal es el desarrollo **Backend con Java**, con una sólida base en lógica de programación, estructuras de datos y modelado de bases de datos relacionales. 

Este proyecto no es solo un CV, sino una muestra de mi capacidad para estructurar código limpio, semántico y profesional desde cero.

---

## 🛠️ Tecnologías y Conceptos Aplicados
- **HTML5 Semántico (Uso de etiquetas estructurales (`<header>`, `<main>`, `<section>`) para accesibilidad y SEO)** 
- **CSS3 Moderno (Mobile Fist)** 
- **Arquitectura de Información (Jerarquía lógica de contenidos para facilitar la lectura técnica)**
